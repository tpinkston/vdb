Captures VDIS traffic (UDP) over a network port/interface with capability
to save captured traffic to a file to be examined or played back afterwards.

Usage:
  vdb COMMAND|-[C|P|L|Q|M] [OPTIONS] [FILE]
  vdb COMMAND|-[C|P|L|Q|M] --examples
  vdb --examples
  vdb --help|-h
  vdb --version|-v

Commands:
  capture    -C  Captures VDIS traffic, optionally stores PDUs in a file
  playback   -P  Plays back VDIS traffic captured in specific file
  list       -L  Lists PDUs in capture file, requires a file argument
  summary    -S  Summarizes capture file contents
  comment    -W  Adds comment to capture file
  uncomment  -U  Remove all or specific comment from capture file
  enums      -M  Prints all known VDIS enumerations in CSV form
  entities       Prints all known VDIS entity types in CSV form
  objects        Prints all known VDIS object types in CSV form

Global options (for all commands):
  -h --help               Print this info
  -m --mono               Suppress colored output
  -q --quiet              Print as little as possible
  -v --verbose            Print more than usual (debugging, -vv extra verbose)
  -w --warnings           Show warnings (suppressed by default)
  -R --suppress           Suppress any error messages
     --examples           Prints example commands
  -V --version            Prints version information

File options (playback, list, summary):
  -p --pcap               Assumes FILE is a 'pcap' file

Network options (capture, playback):
  -6 --ipv6               Use IPv6
  -a --address=ADDR       Use IP address, e.g. 192.168.1.123
  -n --iface=IFACE        Use network interface, e.g. 'eth0'

Filter options (playback, list, summary):
  -r --range=RANGE        Use only PDUs in specified range (indexes)

Filter options (capture, playback, list, summary):
  -H --hostname=H1,...    Include PDUs from specified host(s)
  -e --exercise=E1,...    Include PDUs with exercise number(s) (numeric)
  -E --xexercise=E1,...   Exclude PDUs with exercise number(s) (numeric)
  -t --type=T1,...        Include PDUs with PDU type(s) (numeric)
  -T --xtype=T1,...       Exclude PDUs except PDU type(s) (numeric)
  -f --family=F1,...      Include PDUs with PDU family(s) (numeric)
  -F --xfamily=F1,...     Exclude PDUs with PDU family(s) (numeric)
  -i --ids=I1,...         Include PDUs containing specified entity ID(s)
  -I --xids=I1,...        Exclude PDUs containing specified entity ID(s)

Display options (capture, playback):
  -q --quiet              Print as little as possible
  -s --summarize          Print short PDU summary
  -x --extract            Print PDU data extraction (overrides -s)
  -d --dump               Print PDU hex dump (overrides -s and -x)

Association options (capture, list):
     --associations       Prints only info for associations (sling, mount, etc) 
  -X --extra              Print extra information

Summary command options (summary only):
     --collisions         Print collision info for all entities
     --emissions          Print emission (radar/IFF) info for all entities
     --fires              Print fire/detonation info for all entities
     --lasers             Print designator info for all entities
     --radios             Print radio info for all entities
     --objects            Print object state (point, linear, areal) info
  -A --all                Print all the above info
  -X --extra              Print extra information
