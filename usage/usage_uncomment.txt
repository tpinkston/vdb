Usage:

  vdb uncomment [OPTIONS] FILE [COMMENT#]

Options:

  -h --help                 Print this info
  -v --verbose              Print as much as possible (debugging)
  -w --warnings             Show warnings (suppressed by default)
  -E --errors-off           Do not show any error messages

Examples:

  Removes all comments from file (after confirmation):
  
    $ vdb uncomment capture.vdb

  Removes comment #3 from file (after confirmation):
  
    $ vdb uncomment capture.vdb 3
